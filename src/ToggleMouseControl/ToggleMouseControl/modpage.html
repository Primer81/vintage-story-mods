Mouse control is now toggleable. Stop handbook, world map and other GUIs from controlling the mouse.

 

For those who wish immersive mouse mode was actually immersive. 😜

 

Works with and without immersive mouse mode with slight behavior changes as follows:

  • With immersive mouse mode OFF, most GUIs will auto-enable mouse control when opened and auto-disable mouse control when closed. This mode is useful if you don't want to toggle mouse control each time you open or close a GUI that has an auto-toggle trigger in Vanilla.
  • With immersive mouse mode ON, most GUIs will NOT auto-toggle the mouse control when opened or closed. This mode provides the player with the most control over the mouse.
